# Miniguia de Estudos - NotebookLM

## 1. Contexto e Objetivos
Este caderno temático explora o uso da Inteligência Artificial como ferramenta de aprendizagem ativa.  
Objetivos:
- Entender conceitos fundamentais de IA.
- Aplicar técnicas de engenharia de prompts.
- Criar um guia prático para revisões futuras.

## 3. Engenharia de Prompts e Cicatrizes
- Pergunta inicial: "Explique como a IA pode apoiar estudantes."
- Variações testadas: "IA na educação básica", "IA como tutor virtual".
- Dificuldades: respostas muito genéricas → solução: prompts mais específicos.
- Resultado: resumos detalhados com exemplos práticos.

## 4. Miniguia de Estudo
### Resumos
- IA é um campo multidisciplinar que combina estatística, computação e cognição.
- NotebookLM permite organizar fontes e gerar insights personalizados.

### Glossário
- **Prompt Engineering**: técnica de elaborar perguntas para obter melhores respostas da IA.
- **LLM (Large Language Model)**: modelo de linguagem treinado em grandes volumes de dados.

### Prompts Reutilizáveis
- "Resuma os principais conceitos de IA em 5 tópicos."
- "Explique como aplicar IA em projetos educacionais."
- "Liste vantagens e riscos do uso de IA na aprendizagem."
